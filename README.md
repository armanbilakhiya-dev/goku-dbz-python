# Goku Kamehameha Effect using Python

A real-time computer vision project that simulates Goku's Kamehameha attack using webcam input.

## Features
- Hand tracking using MediaPipe
- Energy ball formation between hands
- Blue glowing beam animation
- Particle effects for realism
- Real-time webcam interaction

## Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy

## How it works
1. Bring both hands together → energy charges
2. Hold position → energy ball grows
3. Release → beam fires horizontally
